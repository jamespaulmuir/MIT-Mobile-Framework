Get info on MIT people, shuttles and classes via text message.

Simply text message your request to 649338 (”MITEDU”) and we'll text message back the information you need.

For a list of keywords and tips, text message "help" to 648338 ("MITEDU")

This service is free from MIT, but carrier charges may apply.

To get started, just text “help” to 648338 (“MITEDU”)

The MIT SMS Service (BETA) includes:

People Directory

Search all or part of a name, email address, or phone number.

Keywords: dir, ppl
For help: dir [help]
Examples:
  • dir hockfield
  • dir 6172531000
  • ppl charles vest

Shuttle Schedule

Get an up-to-the-minute schedule by shuttle name or two-letter abbreviation.

Keywords: bus, shuttle
For help: bus [help]
Examples:
  • bus ce
  • bus cambridge east
  • bus nw
  • bus bus northwest
  • bus shuttle bd
  • bus shuttle boston daytime
Shuttle route abbreviations:
Boston Daytime BD
Boston East BE
Boston West BW
Cambridge East CE
Cambridge West CW
Northwest Shuttle NW
Tech Shuttle TS

Stellar

Get course information and latest announcements by class number.

Keywords: cls, class, stellar
For help: cls [help]
Examples:
  • cls 6.002
  • class 8.03
  • stellar 18.06

How do I use it?

On your mobile device, send a text message to 648338 ("MITEDU"). The text messages must take the format [module keyword] [query]. For instance, to look up class 6.002 in Stellar, text "cls 6.002" or "stellar 6.002". See the details to the left for the keywords for each module.
